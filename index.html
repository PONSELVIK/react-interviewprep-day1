1. WHAT IS VIRTUAL DOM

Virutual Dom [VDOM] is an in-memory representation of real Dom.The representation of UI is kept in memory and synced with Real dom. Its a step between the render function is being called and displaying element on the screen. this entire process is called reconcilation.

2.What is SPA

Single Page Application: Any web application when we click the button or when we try to select any tabs in navbar then the browser page will get reloading that is known as Multi page-application.instead of any changes without reloading the page is known as single-page-application. it is not taking loading time, faster than MPA.

3.DIFFERNCE BETWEEN CLASS AND FUNCIONAL COMPONENT

FUNCTIONAL COMPONENT:

  • It is like a plain javascript function which passed props as arguments and return react element.
  • there is no render menthod.
  • also known as stateless component which accept data and display some form, mainly used for rendering UI
  • Life cycle method not used in functional component.[eg componentDidMount]
  • constructor is not used in functioal component
  • Hooks are simply used to make them statefull.

CLASS COMPONENT

  • It requires render from React.Component and create a render function and returns react element
  • must have render()method returning jsx
  • also known as statefull component.
  • constructor is used to store state
  • life cycle method is used in class component.

4.WHAT DO YOU MEAN BY STATE AND ITS USE IN REACT

State is an object hold some information may change over the lifetime of the component. state is always try to make simple. 

            class comp extends React.Component{
                constructor(props){
                    super(props)

                    this.state ={
                        message : "react"
                    }
                }
                render(){
                    return(
                        div>
                            h1>{this.state.message}/h1>
                        /div>
                )
                }
            }
state is used for internal communication inside react component
state is similar to props but it is private and fully accessed by component.other component can not use without any access.
